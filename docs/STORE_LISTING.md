## 应用名称

- **中文:** GameNest - 局域网联机桌游
- **English:** GameNest - LAN Party Games

## 短描述（80 字以内）

**中文:**
同一个 Wi-Fi 下，开浏览器就能玩的 23 款桌游、卡牌和派对游戏合集。

**English:**
23 board, card, and party games. One server, browser UI, same WiFi — zero setup.

## 完整描述（Google Play 4000 字限制）

### 中文

GameNest 是一个开源、自托管的本地联网联机桌游平台。只需要一台电脑启动服务，同一 WiFi 下的手机、平板、笔记本打开浏览器就能加入，无需安装任何客户端。

**为什么选 GameNest？**

🎲 **23 款游戏，总有适合的**
从经典棋类到派对卡牌，从推理博弈到脑力竞速：
- 经典棋盘：井字棋、五子棋、飞行棋、中国象棋、围棋
- 派对卡牌：UNO、爆炸猫、数字炸弹、抽鬼牌、你画我猜、真心话大冒险
- 推理：达芬奇密码、骗子酒馆
- 扑克：斗地主、大老二、德州扑克
- 桌面策略：魔力桥、大富翁
- 脑力竞速：24点、扫雷竞速、羊了个羊
- 实时对战：贪吃蛇大乱斗、合成大西瓜

📱 **Android App + 浏览器双端支持**
- Android 原生 App，自动启动服务器，即开即玩
- 其他平台通过浏览器访问，无需安装

🤖 **AI 对手**
大部分回合制游戏内置电脑 AI，人不够也能开局。

🌐 **中英文双语**
支持中文和英文界面，一键切换。

🔧 **开源 Apache-2.0 协议**
代码完全开源，可自由修改、分发、自部署。

**使用方式：**
1. 电脑启动服务
2. 同一 WiFi 下打开浏览器扫描二维码或输入 IP
3. 选游戏、开房、开玩

### English

GameNest is an open-source, self-hosted LAN multiplayer tabletop game platform. Start the server on one computer, and any phone, tablet, or laptop on the same WiFi can join instantly through a browser — no app installation required.

**Why GameNest?**

🎲 **23 games for every occasion**
From classic board games to party card games, from deduction to brain racing:
- Classic Board: Tic-Tac-Toe, Gomoku, Flight Chess, Chinese Chess, Go 9x9
- Party Cards: UNO, Exploding Kittens, Number Bomb, Old Maid, Draw & Guess, Truth or Dare
- Deduction: Davinci Code, Liar's Bar
- Poker: Dou Dizhu, Big Two, Texas Hold'em
- Tabletop Strategy: Rummikub, Monopoly
- Brain Racing: 24 Game, Minesweeper Race, Sheep Tile
- Real-time Battle: Snake Battle, Suika Battle

📱 **Android App + Browser**
- Native Android app with built-in Node.js server
- Other platforms join via browser — no install needed

🤖 **AI Opponents**
Most turn-based games include computer AI. Play solo or with a small group.

🌐 **Bilingual (Chinese/English)**
Full UI supports both Chinese and English — switch with one tap.

🔧 **Open Source (Apache-2.0)**
Fully open source. Fork, modify, self-host, or contribute.

**How to play:**
1. Start the server on a computer
2. Scan the QR code or enter the server IP from any device on the same WiFi
3. Pick a game, create a room, start playing

## 截图建议

上架 Google Play Console 需要至少 2 张截图（最多 8 张），建议：

1. **游戏大厅** — 展示精选游戏卡片布局
2. **房间等待页** — 展示玩家列表、准备按钮、扫码功能
3. **某款游戏进行中** — 如 UNO、飞行棋、德州扑克的实际对局画面
4. **游戏设置** — 展示可选配置（限时、牌库选择等）
5. **AI 对局** — 展示电脑玩家参与

尺寸: 至少 320px，最长边不超过 3840px，建议 1080×1920 ( portrait )。
