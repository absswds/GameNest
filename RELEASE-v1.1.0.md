# v1.1.0 — Full Bilingual Support & Visual Polish

## 🇨🇳 中文

### 完整双语支持

- 全部 17 款游戏渲染器支持英文界面 — 大厅点击 🌐 一键切换中/EN
- 大富翁：棋盘地产名、机会卡、错误信息全部双语
- 你画我猜：8 大词库各含约 65 个英文词条
- 游戏目录：23 款游戏均有英文名称、副标题、简介、分类和标签

### 视觉升级

- 英文字体：**Nunito**（圆润现代）
- 中文字体：**LXGW WenKai 落霞孤鹜文楷**（楷体风格，优雅温暖）
- 大富翁棋盘：英文地产名换行优化，火车/电力公司格子不再拥挤

### 修复

- 你画我猜"清除"按钮在画画阶段失效
- 桌面 EXE 启动器端口重试导致进程爆炸
- 电脑玩家名称支持双语（"电脑1" / "Bot 1"）

### 保留中文的内容

- 中国象棋棋子（帅/将/車/馬/炮/象/士/兵/卒）和棋盘标注（楚河/汉界）
- 斗地主牌面标识（大王/小王）
- 合成大西瓜水果名

## 🇺🇸 English

### Full Bilingual Support

- All 17 game renderers now support English UI — switch via the 🌐 button in the lobby
- Monopoly: property names, chance cards, error messages all bilingual
- DrawGuess: 8 word categories with ~65 English entries each
- Game catalog: all 23 games have English names, subtitles, descriptions, categories and tags

### Visual Upgrade

- English font: **Nunito** (rounded, modern)
- Chinese font: **LXGW WenKai** (elegant kai-style, warm and readable)
- Monopoly board: English property name wrapping fixed — railroad/utility cells no longer cramped

### Fixes

- DrawGuess "Clear" button fix during draw stage
- Desktop EXE launcher port retry causing process explosion
- Bot player names now bilingual ("电脑1" / "Bot 1")

### Kept in Chinese

- Chinese Chess piece glyphs (帅/将/車/馬/炮/象/士/兵/卒) and board labels (楚河/汉界)
- Dou Dizhu card identifiers (大王/小王)
- Suika Battle fruit names
